## Todo List
> It is a Todo list app build using JavaScript.

## Description:

It is an app that helps managing all your daily tasks!

## Built With

- Javascript
- Webpack

## How to Run the App Locally
```
   git clone https://github.com/MarvellousUbani/todolist
   cd todolist
   open index.html file

```
